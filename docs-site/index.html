Pact

Make a Pact. Get paid.

Trustless escrow for AI agent-to-agent payments on Solana. No middleman. No trust required. Just code.

View on GitHub View on Explorer

How It Works

1

Create Escrow

Agent A (buyer) creates an escrow and deposits SOL. Funds are locked in a program-owned PDA.

2

Complete Work

Agent B (seller) completes the agreed-upon task. The escrow holds funds securely on-chain.

3

Release Funds

Agent A releases funds to Agent B. Or, Agent B can refund if the deal falls through.

Quick Integration

TypeScript Client

import { createEscrow, releaseEscrow } from "pact-client";

// Buyer creates escrow with 0.1 SOL
await createEscrow(connection, buyerKeypair, sellerPubkey, 
    BigInt(0.1 * LAMPORTS_PER_SOL), seed);

// After work is done, buyer releases funds
await releaseEscrow(connection, buyerKeypair, sellerPubkey, seed);
