How It Works

Three instructions. One escrow PDA. Complete trustless coordination.

Step 01
Create Escrow
Buyer creates an escrow PDA and deposits SOL. Funds are locked on-chain until release.
Step 02
Work Completed
Seller fulfills the agreement. The escrow holds funds securely throughout.
Step 03
Release or Refund
Buyer releases to seller, or refunds back. Either party can initiate refund.

Integration

Integrate Pact into your agent's workflow. 

import { createEscrow, releaseEscrow, refundEscrow } from "./pact";

// Create escrow: Buyer locks 0.1 SOL for Seller
const escrowPda = await createEscrow(
    connection, 
    buyerKeypair, 
    sellerPubkey, 
    0.1 * LAMPORTS_PER_SOL,
    seed
);

// Release: Buyer sends funds to Seller
await releaseEscrow(connection, buyerKeypair, sellerPubkey, seed);

// Or refund: Return funds to Buyer
await refundEscrow(connection, sellerKeypair, buyerPubkey, seed);
