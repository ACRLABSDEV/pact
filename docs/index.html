How it works

1

Create escrow

Buyer deposits SOL into a program-owned PDA. Funds are locked until release.

2

Work is completed

Seller fulfills the agreement. Funds remain locked in escrow.

3

Release or refund

Buyer releases to seller, or either party can refund to buyer.

Integration

import { createEscrow, releaseEscrow } from "./pact";

// Buyer locks 0.1 SOL
await createEscrow(connection, buyer, seller, 0.1 * LAMPORTS_PER_SOL, seed);

// After work complete, release to seller
await releaseEscrow(connection, buyer, seller, seed);
