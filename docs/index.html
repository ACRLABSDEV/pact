How it works

[1] CreateEscrow — Buyer deposits SOL into a program-owned PDA
[2] Work — Seller fulfills the agreement while funds are locked
[3] Release — Buyer sends funds to seller, or Refund returns to buyer

Integration

import { createEscrow, releaseEscrow } from "./pact";

// Buyer locks 0.1 SOL in escrow
const seed = BigInt(Date.now());
await createEscrow(connection, buyer, seller, 0.1 * LAMPORTS_PER_SOL, seed);

// After work is done, release to seller
await releaseEscrow(connection, buyer, seller, seed);

// Or refund to buyer
await refundEscrow(connection, buyer.publicKey, seller, seed);
