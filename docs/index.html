On-chain escrow for agent payments

Lock funds in a PDA. Complete work. Release or refund. No intermediary.

pact demo — agent-to-agent escrow
# Agent A (buyer) creates escrow for Agent B (seller)
agent-a $ pact create --seller AgentB --amount 0.1
Creating escrow...
✓ Escrow created
PDA: 82VUPuMmx9rjBhKSYAJECsMTmdQ5KMkiJh5GWGS3UUS7
Amount: 0.1 SOL
Status: Active
 
# Agent B completes the work...
 
# Agent A releases funds to Agent B
agent-a $ pact release --escrow 82VUPuMm...
Releasing funds...
✓ Funds released to AgentB
Tx: 4vJ9...kL2m
Escrow complete.
Program ID S64L6x9bZqDewocv5MrCLeTAq1MKatLqrWfrLpdcDKM

How it works

[1] CreateEscrow — Buyer deposits SOL into a program-owned PDA
[2] Work — Seller fulfills the agreement while funds are locked
[3] Release — Buyer sends funds to seller, or Refund returns to buyer

Integration

# Fetch the skill
$ curl -s https://acrlabsdev.github.io/pact/SKILL.md

# Or use the TypeScript client
import { createEscrow, releaseEscrow } from "./pact";

await createEscrow(connection, buyer, seller, 0.1 * LAMPORTS_PER_SOL, seed);
await releaseEscrow(connection, buyer, seller, seed);
